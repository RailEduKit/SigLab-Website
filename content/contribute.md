# Contribute

Contributions are welcome across all repositories in the SigLab constellation.

## Where to contribute

- Website structure and portal content: `SigLab-Website`
- 3D model source and technical build docs: `SigLab-3Dmodels`
- Handouts and print artifacts: `SigLab-handouts`
- Paper prototype sheets: `SigLab-Papercut`

## Contribution workflow

1. Open an issue in the relevant repository.
2. Propose a small focused change.
3. Include evidence (photo, rendered preview, or PDF diff) when applicable.
4. Keep licensing compatible with CC BY-SA 4.0 project goals.
