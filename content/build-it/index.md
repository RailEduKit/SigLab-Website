# Build it

Construction source material lives in dedicated repositories. This portal keeps one stable entry point and links to canonical files.

## 3D and wooden setup

- Main source: [SigLab-3Dmodels](https://github.com/RailEduKit/SigLab-3Dmodels)
- Includes OpenSCAD, STL, rendered images, and component-level documentation.

## Paper prototype setup

- Main source: [SigLab-Papercut](https://github.com/RailEduKit/SigLab-Papercut)
- Includes printable paper components and templates.
