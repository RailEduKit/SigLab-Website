# Bill of materials

This page is the website-level procurement map. Detailed and versioned BoM files stay in the specialist repositories.

## Canonical BoM sources

- Handouts and teaching print materials: `SigLab-handouts`
- Paper prototype materials: `SigLab-Papercut/bill_of_materials.pdf`
- Wooden and printed components: `SigLab-3Dmodels/documentation/materials.md`

## Minimal starter set

- Track elements and base rolling stock
- Signals and route/locking markers
- Printed handouts for rules and scenarios
- Session protocol sheets

For direct files generated by the build pipeline, go to [downloads](../downloads.md).
