# Teach with it

This section is intentionally owned by the website repository and focuses on facilitation guidance rather than raw component files.

## Facilitation baseline

- Prepare a 3 x 2.5h session rhythm.
- Define learner roles before play starts.
- Use explicit debriefing slots after each scenario block.
- Capture observations in a shared protocol template.

## Near-term additions

- Typical learner misconceptions and intervention patterns.
- Debriefing method templates.
- Mapping to railway engineering and vocational curricula.
